
# Behavioral Alpha Engine

## AI-Powered Behavioral Intelligence Platform

This project analyzes behavioral finance biases among affluent investors and builds an interactive wealth advisory intelligence system.

### Features
- Behavioral Bias Detection
- Investor Segmentation
- Advisory Intelligence
- Portfolio Risk Analysis
- Behavioral Scorecards
- AI Advisory Recommendations

### Tech Stack
- Python
- Streamlit
- Plotly
- Scikit-learn
- Pandas

### Deployment
Hosted using Streamlit Cloud.
